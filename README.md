# HTTP Devices

Thinger.io plugin for handling generic HTTP devices.

<img src="https://okli.in/wp-content/uploads/2021/04/resticon-removebg-preview.png" width="150px">

## Plugin features

* Automatic device and bucket provisioning for new devices included. 
* Store device data automatically in data buckets so it can be easily used from the console.
* Provide support por defining custom request data processing on NodeJS.
* Provide support for defining custom response data processing on NodeJS.

## Documentation

Check the plugin [documentation](https://docs.thinger.io/plugins/http-device) for more information.
